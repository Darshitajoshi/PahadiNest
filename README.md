# 🏔️ PahadiNest

PahadiNest is an AI-powered homestay discovery platform for Uttarakhand. It helps travelers find the best homestays with personalized recommendations, nearby attractions, and a seamless booking experience.

## 🚀 Tech Stack

### Frontend

* React.js
* Vite
* Tailwind CSS
* React Router DOM

### Backend

* Node.js
* Express.js
* CORS
* dotenv

### Database *(Coming Soon)*

* MongoDB

### AI *(Coming Soon)*

* Google Gemini API

---

## ⚙️ Backend Setup

```bash
cd backend
npm install
npm run dev
```

Create a `.env` file:

```env
PORT=5000
```

Backend runs on:

```
http://localhost:5000
```

---

## 💻 Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on:

```
http://localhost:5173
```

---

## 📡 API Endpoints

* GET `/api/homestays`
* GET `/api/homestays/:id`
* POST `/api/homestays`
* PUT `/api/homestays/:id`
* DELETE `/api/homestays/:id`
* GET `/api/homestays/search?q=keyword`
